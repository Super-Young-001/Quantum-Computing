### Main Problem

**Qubit Error** is the biggest problem in Quantum Computing.

Quantum Error Correction is Expensive.

- In the near-term we must use Noisy Intermediate Scale Quantum Computers(NISQ).
- Fault-Tolerant Quantum Computing(long term).

### NISQ Model Of Computer 

- Quantum program - Compiler - Quantum Executable - Quantum Computer(Repeat Execution for N Trails) - 1010(output)
- Despite qubit errors, correct outcome can be inferred on NISQ

We can develop software transformation to reduce the impact of hardware errors on Quantum Computer.

The Problem of limited connectivity - Existing compiler policies focus on minimizing SWAPs.
